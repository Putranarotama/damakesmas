# DAMAKESMAS Typography Guide
 
## Font Families
| Peran | Font | CSS Variable | Penggunaan |
|-------|------|-------------|------------|
| Display | `Poppins` | `--font-display` | Heading, hero, section title, tombol |
| Body | `Poppins` | `--font-body` | Body text, tabel, badge, nav link |
 
 
## Type Scale
| Token CSS | Font | Ukuran | Weight | Line-Height | Penggunaan |
|-----------|------|--------|--------|-------------|------------|
| `.text-display` | Poppins | 48px / 3rem | 800 | 1.1 | Hero headline utama |
| `.text-h1` | Poppins | 36px / 2.25rem | 700 | 1.2 | Judul halaman |
| `.text-h2` | Poppins | 24px / 1.5rem | 700 | 1.3 | Section title |
| `.text-h3` | Poppins | 20px / 1.25rem | 600 | 1.4 | Card title, sub-section |
| `.text-body-lg` | Poppins | 16px / 1rem | 400 | 1.7 | Body text utama |
| `.text-body` | Poppins | 14px / 0.875rem | 400 | 1.6 | Tabel, teks sekunder |
| `.text-sm` | Poppins | 12px / 0.75rem | 400 | 1.4 | Caption, label, footer |
| `.text-stat` | Poppins | 28px / 1.75rem | 700 | 1 | Angka statistik card |
 
## Font Weight
| Nilai | Nama | Penggunaan |
|-------|------|------------|
| 400 | Regular | Body text, deskripsi, placeholder |
| 500 | Medium | Nav link, badge label |
| 600 | SemiBold | Sub-heading, tombol sekunder, label form |
| 700 | Bold | Section title, card title, heading |
| 800 | ExtraBold | Hero title, brand name, tombol CTA |
 
## Letter Spacing
| Konteks | Nilai | Contoh |
|---------|-------|--------|
| Brand / Navbar | `0.05em` | "DAMAKESMAS" |
| Section label (uppercase) | `0.08em` | "AKSES CEPAT", "USERNAME" |
| Nav link | `0.04em` | Menu item aktif |
| Body | `0` | Paragraf, tabel |
