# DAMAKESMAS Spacing System

## Base Unit: 4px

| Token | Value | Tailwind | Usage |
|-------|-------|----------|-------|
| xs | 4px | p-1 / gap-1 | Icon padding, tight badges |
| sm | 8px | p-2 / gap-2 | Inner card padding tight |
| md | 16px | p-4 / gap-4 | Component padding standard |
| lg | 24px | p-6 / gap-6 | Card padding, section internal |
| xl | 32px | p-8 / gap-8 | Section padding vertical |
| 2xl | 48px | p-12 / gap-12 | Section spacing major |
| 3xl | 64px | p-16 / gap-16 | Hero padding, page sections |
| 4xl | 96px | p-24 | Hero vertical padding |

## Border Radius
| Token | Value | Usage |
|-------|-------|-------|
| sm | 6px | Badges, tags |
| md | 10px | Buttons |
| lg | 12px | Cards |
| xl | 16px | Hero cards, modals |
| 2xl | 24px | Search bar |
| full | 9999px | Pill badges |

## Shadow
| Token | Value | Usage |
|-------|-------|-------|
| card | `0 2px 8px rgba(0,0,0,.06)` | Default card |
| card-hover | `0 6px 20px rgba(0,0,0,.10)` | Card hover |
| nav | `0 2px 12px rgba(0,0,0,.08)` | Navbar |
| dropdown | `0 10px 30px rgba(0,0,0,.12)` | Dropdowns |
